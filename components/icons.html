FontCustom Icon Build Process

View on GitHub open_in_new
  • Designer builds out the needed icon in SVG format
    • Dimensions 250px X 250px
    • Color #000000
    • Solid shapes and outlined fonts only
  • Designer provides these icons to the assigned team developer to compile the font
    • Blue Connect Mobile - Seth Rhodes
    • Blue Connect 2.0 - Anand Krishnankutty
    • Blue Connect 1.0 - This team uses individual assets as well as the 2.0 font custom files
    • Smart Desktop - N/A This team utilizes the Material Design icon font only
    • Experience Health - TBD
  • Developer recompiles the font following the instructions on this GitHub page
  • Developer adds the new version to the following repositories in GitHub:
    • Blue Connect Mobile Icons
    • Blue Connect Web Icons
